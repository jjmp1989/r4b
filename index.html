Rationality 4 Business

Rationality For Business ERP es un software open source dirigidos a PYMES dándoles la posibilidad de registrar sus gestiones empresariales de una manera fiable y rápida. Con esta aplicación puede administrar sus operaciones comerciales desde la compra a la venta de mercaderías y servicios.

¡Descargue R4B y gestione su negocio!

Con Rationality For Business tendrá una herramienta que le facilitará la administración de sus negocios en varios departamentos empresariales. Este software le permitirá gestionar varias compañías en una sola aplicación, dar de alta a socios comerciales, gestionar compras y ventas, así como llevar a cabo el control del inventario de mercaderías y servicios ofertados.

En esta web podrá descargar la aplicación para diferentes plataformas, acceder a su código y modelo de BBDD, y consultar el manual del software para su instalación y uso.

Características

A continuación, se muestran algunas características más destacables de R4B.

Gestión Empresarial

Posibilidad de registrar datos sobre compañías propias y los socios comerciales con los que se relacionan las mismas. Establezca si un socio comercial es cliente, proveedor o ambos.

Compras

Registre en sus compañías los pedidos y controle su evolución para las operaciones de compras tanto de mercaderías como de servicios con sus proveedores.

Inventario

Lleve el registro y control los inventarios de mercaderías y servicios. Automatice el recuento de inventario así como el precio unitario (PMP) de cada mercadería basándolo en su coste de compra.

Ventas

Realice para sus diferentes compañías los pedidos de ventas de sus clientes para grabar toda la información referente a la misma para un mejor control.

Guía de Instalación

Requisitos

Para poder usar esta aplicación se necesitarán los siguientes requerimientos previos:

  • Tener instalada la JVM versión 11 o superior.
  • Descargar las librerías JAVAFX.
  • Tener configuradas las variables de entorno.
  • MySQL Server y MySQL-Workbench. Este último es opcional pudiendo usarse otro software cliente de SQL similar como DBeaver.
Descarga de la Aplicación

En primer lugar se debe descargar la aplicación para el sistema operativo deseado en la sección anterior destinada para tal fin. En esta sección podrá descargar la aplicación para Linux y Windows, así como acceder a su código fuente.

Tras su descarga, ir a la carpeta de la BBDD, el abrir el modelo en MySQL-Workbench y crear la BBDD a partir del mismo. Entonces, abrir y ejecutar en el mismo cliente los scripts para la creación de procedimientos almacenados, vistas y triggers.

Para ejecutar la aplicación en Windows, ir a la consola de Windows, cambiar de directorio hasta la carpeta de la app y ejecutar la siguiente instraucción:

java --module-path "ruta_a_librerias_javafx\lib" --add-modules javafx.base,javafx.controls,javafx.fxml,javafx.graphics -jar r4b.jar

Gestión Empresarial

Compañías

R4B ERP le permite gestionar diferentes compañías en módulo de Gestión Empresarial. Para ello siga los siguientes pasos:

  • En el menú inicial pulse en Gestión Empresarial.
  • Haga click en el submenú inferior de la parte izquierda en Empresas y acceda a la lista de empresas.
  • Para crear una nueva empresa pulse en el botón verde 'Nueva compañía' y rellene el formulario con los datos requeridos.
  • Si desea modificar una compañía, seleccione la compañía a modificar y haga click en el botón púrpura 'Modificar compañía'. Entonces, modifique los datos que necesite en el formulario.

Este módulo le permitirá llevar el control del inventario de mercaderías y servicios, así como su coste unitario. Para acceder al módulo del inventario, en el menú principal presione el botón con el mismo nombre. De esta manera, se accede a la pantalla inicial del inventario.

Para visualizar la lista de los tipos de servicios presione el botón con dicho nombre en el submenú de la parte inferior izquierda. En la siguiente imagen puede visualizar dicha pantalla.

Para crear un tipo de servicio pulse en el botón verde Nuevo Tipo de Servicio y rellene el formulario.

Para modificar un tipo de servicio existente pulse en el botón verde Modificar Tipo de Servicio y modifique los datos del formulario.

Servicios

Para visualizar la lista de servicios así como para insertar o modificar los mismos, se procede de manera similar como con los tipos de servicios.

Ventas

Con el módulo de ventas podrá realizar pedidos de ventas de mercaderías y servicios, además de registrar las salidas o finalizaciones de los mismos.

Pedidos de Servicios
  • Para realizar un pedido de servicio, en el módulo de Ventas vaya a Pedidos Servicios. Entonces pasará a la lista de pedidos de ventas de servicios.
  • Para visualizar un pedido, se debe seleccionar y luego pulsar en el botón Ver pedido.
  • Si desea crear un pedido, en la pantalla de la lista pulse en el botón Nuevo Pedido de Vtas. Servicios.
  • Si lo que desea es modificar un pedido ya existente, seleccione el pedido a actualizar y pulse en el botón Modificar Pedido de Vtas. Servicios.

Seguidamente, se muestran las imágenes de las acciones anteriores.

Finalización de Servicios

Las acciones para finalizar los servicios funcionan de manera similar a la realización de pedidos con algunas diferencias como que al crear una línea solo se podrá elegir uno de los servicios del pedido de ventas asociado.
